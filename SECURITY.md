# Security Policy

## Reporting a Vulnerability

If you discover a security issue in `mrcorpt-games-zen`, please report it so we can address it responsibly.

Preferred reporting methods:

- Use GitHub Security Advisories if available.
- Open a private issue labeled with a security concern.

Do not disclose the issue publicly until it has been fixed.

## Supported Versions

- `master` is the current supported branch.

## Security Response

We take security reports seriously and will respond as quickly as possible. After a report is received, we will:

1. Acknowledge receipt.
2. Investigate the report.
3. Fix the issue and release a patched version.
4. Notify the reporter when the issue has been addressed.

## Safe Disclosure

Please provide enough information for us to reproduce the issue, including:

- A clear description of the vulnerability.
- Steps to reproduce.
- Any relevant environment or configuration details.

If you prefer, you may also contact the maintainers directly through the repository.
